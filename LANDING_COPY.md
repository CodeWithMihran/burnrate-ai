## FAQ

### 1. Do I need to sign up before using it?

No. The audit is free to use without login. You only see the email capture after the report is generated.

### 2. What tools does it support?

The audit is built for popular startup AI tools including ChatGPT, Claude, Cursor, GitHub Copilot, Gemini, and API-direct workflows.

### 3. Is this trying to sell me something?

The first job is to produce a useful audit. If your stack has meaningful savings potential, Credex may be a relevant next step, but the report should still be valuable on its own.

### 4. What if my stack is already optimized?

Then the audit should say so. The product is designed to avoid inventing fake savings just to sound impressive.

### 5. How accurate are the recommendations?

Recommendations are based on current pricing inputs and explicit rules, not vague model guesses. The AI is only used to summarize the findings in plain language.
